# Erasmus Hustlers
Erasmus Hustlers is a web platform designed to conect owners of different spaces with people that are searching for a place to celebrate meetings, parties, birthdays or social reunions. 

User stories:

As a user that wants to throw a party, I want to be able to filter rooms by capacity and price per hour to find the appropiate place adjusted to my budget.

As a local owner I want to be able to show photos of my room, adding rules and some restrictions in order to maintain the room in good conditions.

As an interested client I would like to see a calendar with in real time disponibility to know if I can book the local in the date and hour I want without having to make a phone call.
